Tokens système MD3 — valeurs SNCF

Angular Material 20 expose les couleurs via des tokens CSS --mat-sys-*. core-md3.scss injecte les valeurs SNCF exactes via $overrides.

@for (token of tokens; track token.name) {
{{ token.name }} {{ token.value }} {{ token.description }}
}
