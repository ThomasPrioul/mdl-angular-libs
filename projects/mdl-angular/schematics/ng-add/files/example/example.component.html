Exemple de page

Information sur le thème

Ce projet est un exemple de page utilisant le thème mdl-angular.

le header et la barre de navigation verticale se situent dans src/app/components/layout

Pour ajouter un nouvel item dans la barre de navigation, ajouter un élément dans menus se trouvant dans src/app/components/layout/vertical-nav/vertical-nav.component.ts

src/app/components/layout/vertical-nav/vertical-nav.component.ts
              
                
                protected menus: SideMenu[] = [
                {
                  text: 'Accueil',
                  link: '/home',
                  icon: 'home',
                  children: [
                    {
                      text: 'Exemple',
                      link: '/home/example',
                      icon: 'table_rows',
                    },
                  ],
                },
                ];

Les composants instalés:

  • mdl-angular
  • Angular Material
  • Tailwind
